Pc nuevo configurar



revisar el wsl
ver si esta instalado
wsl --status

si ya esta instalado usamos
wsl --update

instalar docker desktop

abrir docker desktop antes de levantar el backend

revisar si en la carpeta backend hay un archivo llamado ".env" creado
debe contener lo sgte:
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=ucn_user
DB_PASSWORD=ucn_pass
DB_DATABASE=proyecciones_db

luego se debe levantar el contenedor
docker compose up -d
ver si funca
docker ps

luego ahi recién
cd backend
npm install



1ero el backend

iniciar backend
ingresar a backend/src en una terminal y ahi poner el comando
npm run start:dev


después el frontend
iniciar frontend
ingresar a Frontend/src en una terminal y ahi poner el comando
npm run dev


abrir la pagina
http://localhost:5173/

y debería abrir


En resumen
Paso	Acción						Comando
1	Clonar repo					    git clone
2	Instalar backend				npm install
3	Instalar frontend				npm install
4	Actualizar WSL				    wsl --update
5	Revisar docker-compose.yml		—
6	Limpiar contenedores			docker compose down -v
7	Levantar base de datos			docker compose up -d
8	Ejecutar backend				npm run start:dev
9	Ejecutar frontend				npm run dev
10	Probar conexión				    localhost:5173



una vez que ya este instalado se puede iniciar normalmente:

-abrir docker desktop

-iniciar backend 
npm run start:dev

-luego el frontend
npm run dev

y funca
